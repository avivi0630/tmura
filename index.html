תמורה
V2.0 Media Architecture

כל המדיה שלך.
במקום אחד.

תמורה היא סוויטת כלים עוצמתית לעיבוד קבצים, המשלבת בינה מקומית עם ממשק אינטואיטיבי וחיבור ישיר לדפדפן.

הורד את הגרסה החדשה התקן תוסף Chrome

יכולות המערכת

כל מה שצריך לעבודה מקצועית עם קבצים

🖼️

אופטימיזציית תמונות

  • כיווץ נפח חכם ללא פגיעה נראית בעין
  • שינוי רזולוציה ומימדים בקבוצה
  • תמיכה מלאה ב-WebP ו-HEIC מודרניים
  • יצירת קבצי ICO רב-שכבתיים
🎬

עיבוד וידאו מקצועי

  • המרת פורמטים מהירה מבוססת FFmpeg
  • כיווץ וידאו מותאם לשיתוף ברשת
  • הפקת GIF מכל סרטון בלחיצת כפתור
  • ניצול ריבוי ליבות (Threads) לביצועים מרביים
🎵

מרכז האודיו

  • עריכת Metadata (תגיות ID3) מלאה
  • החלפה וחילוץ תמונות עטיפה (Cover Art)
  • המרת קבצים באיכות גבוהה (Lossless support)
  • תמיכה בפורמטים MP3, FLAC, WAV ועוד
📄

כלי PDF מתקדמים

  • איחוד מהיר של מספר קבצים למסמך אחד
  • הוספת קווי חיתוך (Crop Marks) לדפוס
  • המרת דפי PDF לתמונות או לטקסט
  • תצוגה מקדימה של דפים (Thumbnails)
🌐

אינטגרציית דפדפן

  • כיווץ תמונות ישירות מהרשת ללוח העריכה
  • מערכת Native Messaging לחיבור מאובטח
  • תמיכה ב-Drag & Drop מאתרים ישירות לתוכנה
  • מצב Gmail ייעודי לייעול עבודה בדוא"ל
🔒

פרטיות וביצועים

  • 100% עיבוד מקומי - שום קובץ לא עולה לענן
  • תמיכה מלאה בעברית, אנגלית ואידיש
  • עבודה בשרת רקע (Background Mode)
  • ממשק Apple-Style נקי ונוח

מנוע עיבוד

Python 3.11 & FFmpeg

50+

פרטיות

Local Processing Only

תאימות
